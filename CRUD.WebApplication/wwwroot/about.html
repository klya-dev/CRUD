О нас

Если мой пользователь хочет использовать мой API, ему надо:
  • Зарегистрироваться
  • Купить премиум
  • Получить API-ключ
  • При запросе на создание статьи приложить данные для статьи и этот ключ
Почти никакого смысла от этого нет, если только пользователь не написал свой удобный, по его мнению, клиент, с которого он пишет статьи и отправляет мне на сайт
Ну вот личная неприязнь к моему клиенту ;)
Если мой пользователь хочет использовать софт для отправки одной статьи на все популярные платформы, ему надо:
  • Зарегистрироваться
  • Купить премиум
  • Получить одноразовый API-ключ (на одну статью)
  • Обратиться к стороннему сервису, написать статью, приложить одноразовый ключ
  • Нажать кнопку отправить
  • Сторонний сервис, отправляет мне запрос с данными для статьи и одноразовый ключ
Что-то похожее он делает и для других платформ, скорее всего таких API у них нет, но это уже не дело моего лора ;)
